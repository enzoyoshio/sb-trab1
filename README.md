## instruções para executar o montador

Primeiramente você deve estar na pasta ```/src``` do projeto.

Use o comando:

``` make montador ```

Após isso, use o comando:

``` ./montador -x program ```

Em que "x" tem que ser p, m ou o e "program" é o nome do código fonte sem a extensão asm.
